¡Bienvenido a mi repositorio destinado a  la asignatura **Programación III**!  
Este proyecto contiene una serie de ejercicios prácticos divididos temáticamente en carpetas. El objetivo es dominar los conceptos fundamentales del lenguaje **C# (.NET)**, enfocándose en la declaración de variables, cálculos matemáticos, estructuras básicas de la consola y buenas prácticas de codificación.

---

## 📂 Organización del Proyecto

El repositorio está segmentado en "Módulos" o "Programas", donde cada uno aborda un tipo de dato o concepto de C# a través de al menos 5 variantes o situaciones de la vida cotidiana.

| Directorio | Temática Principal | Situaciones abordadas |
| :--- | :--- | :--- |
| 📁 **`Programa00` / `01`** | **Introducción** | Hola mundo, entrada/salida de datos por consola (`Console.WriteLine` / `ReadLine`). |
| 📁 **`Programa02`** | **Números Enteros (`int`)** | Cálculo de gastos de supermercado, reparto equitativo de unidades, planes de ahorro y sumatorias de puntajes. |
| 📁 **`Programa03`** | **Números Decimales (`double` / `decimal`)** | Promedio de notas académicas, cálculos de áreas geométricas, casas de cambio de divisas y descuentos. |
| 📁 **`Programa04`** | **Cadenas y Letras (`string` / `char`)** | Formateo de nombres completos, creación masiva de identificadores/emails, uso de iniciales. |
| 📁 **`Programa05`** | **Valores Lógicos (`bool` & `Random`)** | Simulación pseudoaleatoria del sistema (ej. encendido de luces, activación de alarmas, pronóstico de lluvias), sin estructuras condicionales (if). |
| 📁 **`Programa06`** | **Constantes (`const`)** | Valores inmutables aplicados a fórmulas matemáticas (PI), reglas de negocios, límites de velocidad y restricciones de seguridad de acceso (Login). |
| 📁 **`Programa07`** | **Módulo Financiero y Formateo** | Aplicación directa de porcentajes fijos a situaciones fiscales (Propinas, impuestos digitales, comisiones de plataformas dadas). Demostración del formateo estético para consolas (`.ToString("0.00")`). |

## 🚀 Cómo ejecutar los programas

Cada proyecto es una aplicación de consola independiente. Para correr cualquiera de las carpetas:

1. Abrí tu consola o terminal (Simbolo de Sistema, PowerShell o Git Bash).
2. Navegá por carpetas usando `cd` hasta entrar en el proyecto que querés probar.  
   `cd Programa07`
3. Ejecutá el programa con el compilador (.NET CLR) usando el comando:
   ```bash
   dotnet run
   ```

*(En la mayoría de los programas, presionar `ENTER` te permitirá navegar de una variante a otra).*
